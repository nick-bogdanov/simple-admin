Для запуска проекта установите модули

В корне проекта вводим следующие команды

модули для бекенда
npm install

для фронт енда
cd ./frontend
bower install

Для Бд необхимо иметь установленную MongoDB

Для подключения к БД измените настройки в файле:
config/default.json
поле db - напишите свой путь к базе, или оставьте как есть

Для запуска проекта в корне проекта набираем

node ./server/bin/www
